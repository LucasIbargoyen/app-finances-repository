# App Personal Finances #
**Escopo**
- App focado no monitoramento de finanças pessoais do usuário.

**Requisitos Funcionais**
RF01 - O usuário poderá colocar dados sobre a sua renda mensal.
RF02 - O usuário poderá colocar dados sobre os seus gastos fixos.
RF03 - Conforme o usuário for adicionando mais gastos fixos, o app avisará que ele pode estar se comprometendo financeiramente dependendo da sua renda mensal.
RF 04 - O usuário poderá colocar manualmente gastos diários que não sejam fixos.
